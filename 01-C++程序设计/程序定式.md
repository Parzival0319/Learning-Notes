# 冒泡排序
```C
for(int i=0;i<cnt-1;i++){
		for(int j=0;j<cnt-i-1;j++){
			if(result[j]<result[j+1]){
				int t=result[j];
				result[j]=result[j+1];
				result[j+1]=t;
			}
		}
	}
```

# 素数判断
```c
int isprime(int a){
	if(a<=1) return 0;
	if(a==2) return 1;
	if(a%2==0) return 0;
	for(int i=3;i*i<=a;i+=2){
		if(a%i==0) return 0;
	}
	return 1;
}
