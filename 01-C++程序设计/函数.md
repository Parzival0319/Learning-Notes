# sprintf 用法笔记

`sprintf` 是 C 标准库函数，用于将格式化数据输出到字符串（而非屏幕）。

## 函数原型

```c
int sprintf(char *str, const char *format, ...);
```
## 常见用途

1. 整数转字符串
将整数按十进制转换为字符串。
```c
int num = 123;
char buf[20];
sprintf(buf, "%d", num);   // buf 内容为 "123" 
```
2. 浮点数转字符串
可以控制小数位数。
```c
double pi = 3.14159;
char buf[20];
sprintf(buf, "%.2f", pi);  // buf 内容为 "3.14" 
```
3. 拼接字符串
将多个字符串或变量组合成一个字符串。
```c
char name[] = "Alice";
int age = 20;
char buf[50];
sprintf(buf, "Name: %s, Age: %d", name, age);
// buf 内容为 "Name: Alice, Age: 20"
```

# strlen 用法笔记
`strlen`是 C 语言自带的函数,求字符串 s 的长度（有多少个字符）
比如：
```c
char s[] = "abc";
int len = strlen(s);   // len = 3
```
