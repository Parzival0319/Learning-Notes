# 字符是如何转化为数字的？
字符 '0' 到 '9' 在 ASCII 码中是连续排列的，其 ASCII 值分别为 48 到 57。
因此，要将一个数字字符（如 '5'）转换为对应的整数值，可以用 字符减去 '0' 实现：

` int digit = res[i] - '0'; `     // 例如 '5' - '0' = 53 - 48 = 5

# 一行输入、回车就停止
  事实上，scanf("%s") 会自动以空格 / 回车分隔，读到单词成功会返回 1，读到回车返回 EOF，循环自动停止。
   ```c
     while(scanf("%s", word[cnt]) == 1) {
        cnt++;
    } // 最优循环(依赖 EOF)
  ```
  通用答案
  ```c
 char line[1000];   // 足够大，存一整行

    // ✅ 【唯一正确】回车就停止，绝不卡死
    fgets(line, 1000, stdin);   //无法得知存储个数
```
